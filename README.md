# ETF Strategy Comparison App

Compare leveraged ETFs like SSO, TQQQ, QLD using:

- Lump Sum
- Dollar Cost Averaging (DCA)

Built with Streamlit for interactive web-based analysis.

## How to Run

1. Install requirements:

```bash
pip install -r requirements.txt
```

2. Run the app:

```bash
streamlit run app.py
```
3. Open the public URL provided by Streamlit Cloud if deployed.
